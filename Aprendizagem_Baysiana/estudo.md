# Glossário

## Correção de Laplace:
Uma técnica utilizada para evitar probabilidades zero em modelos de classificação, adicionando uma contagem fictícia aos dados para garantir que o modelo Naive Bayes possa generalizar melhor.

## Matriz de Confusão:
Uma ferramenta utilizada para visualizar o desempenho de um modelo de classificação, mostrando o número de previsões corretas e incorretas divididas por cada classe.

## Naive Bayes:
Um algoritmo de aprendizado de máquina baseado no Teorema de Bayes, que assume independência condicional entre as características para realizar classificações.

## Probabilidade a Priori:
A probabilidade inicial de uma hipótese antes de novos dados serem considerados, essencial para evitar vieses em dados desbalanceados.

## Probabilidade Condicional:
A probabilidade de um evento ocorrer dado que outro evento já ocorreu, fundamental para o cálculo no Teorema de Bayes.

## Recall:
Uma métrica de avaliação de modelos de classificação que mede a capacidade do modelo de identificar corretamente todas as instâncias relevantes.

## Score de Crédito:
Uma avaliação numérica da capacidade de crédito de uma pessoa, que pode ser predita usando modelos como o Naive Bayes.

## Teorema de Bayes:
Um princípio fundamental em estatística que permite atualizar a probabilidade de uma hipótese com base em novos dados.

## Teste de Avaliação:
A fase em que um modelo de aprendizado de máquina é avaliado em dados não vistos para medir seu desempenho e ajustar suas previsões.

## Treinamento de Modelos:
O processo de ajustar um modelo de aprendizado de máquina aos dados de treinamento para que ele possa fazer previsões precisas em novos dados.
