# Glossário

## Análise Bivariada:
Técnica de análise de dados que examina a relação entre duas variáveis, permitindo identificar padrões e correlações que podem ser úteis para previsões e tomadas de decisão.

## Análise Univariada:
Técnica de análise de dados que se concentra em uma única variável, utilizando estatísticas descritivas para entender sua distribuição, tendência central e dispersão.

## Estatísticas Descritivas:
Conjunto de técnicas utilizadas para descrever e resumir as características principais de um conjunto de dados, incluindo medidas de tendência central (média, mediana) e dispersão (desvio padrão, variância).

## Outliers:
Pontos de dados que se desviam significativamente dos demais, podendo indicar variabilidade extrema, erros de medição ou fenômenos novos e interessantes.

## Pré-processamento:
Etapa inicial de preparação dos dados para análise, que inclui limpeza, transformação e normalização dos dados para garantir que estejam em um formato adequado para análise e modelagem.
