# Glossário

## Coeficiente de Determinação (R²):
Uma medida estatística que indica a proporção da variância na variável dependente que é previsível a partir das variáveis independentes no modelo de regressão.

## Equação da Reta:
A fórmula Y = mX + b, usada na regressão linear simples para modelar a relação entre uma variável dependente (Y) e uma variável independente (X), onde m é a inclinação e b é o intercepto.

## Matriz de Correlação:
Uma tabela que mostra os coeficientes de correlação entre várias variáveis, usada para identificar e quantificar a relação entre elas em um conjunto de dados.

## Modelo de Regressãp Linear Múltipla:
Uma extensão da regressão linear simples que inclui múltiplas variáveis independentes para prever o valor de uma variável dependente.

## Pré-processamento de Dados:
O conjunto de técnicas aplicadas aos dados brutos para prepará-los para a análise, incluindo limpeza, transformação e normalização dos dados.

## Regressão Linear Simples:
Uma técnica de modelagem estatística que utiliza a equação da reta para prever o valor de uma variável dependente com base em uma única variável independente.

## Validação de Modelos:
O processo de avaliar a precisão e a eficácia de um modelo preditivo usando dados históricos antes de aplicá-lo em cenários reais.
