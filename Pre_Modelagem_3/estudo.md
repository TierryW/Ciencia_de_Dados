# Glossário

## Associação:
Refere-se a uma relação estatística entre duas ou mais variáveis, onde a presença de uma variável está relacionada à presença de outra, mas sem implicar causalidade.

## Causalidade:
Descreve uma relação onde uma variável diretamente influencia outra, indicando uma relação de causa e efeito, ao contrário da correlação, que não implica causalidade.

## Label Encoding:
Uma técnica de pré-processamento de dados categóricos que transforma categorias em números inteiros, permitindo que modelos matemáticos processem dados categóricos.

## Multicolinearidade:
Ocorre quando duas ou mais variáveis independentes em um modelo de regressão estão altamente correlacionadas, o que pode causar instabilidade nos coeficientes do modelo.

## One-Hot Encoding:
Uma técnica de pré-processamento que transforma dados categóricos em vetores binários, criando uma coluna para cada categoria, onde apenas uma coluna é marcada como 1 (ativa) e as demais como 0.

## SMOTE (Synthetic Minorrity Over-sampling Technique):
Uma técnica de balanceamento de dados que cria exemplos sintéticos da classe minoritária para lidar com conjuntos de dados desbalanceados, aplicando o balanceamento apenas na base de treino.
